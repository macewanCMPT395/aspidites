IPython Turtle Functionality.

View project onGitHub

If you want to add to, or modify this project here are a few things that will be useful to keep in mind.

The changes are primarily located in:
"IPython/html/static/notebook/js/outputarea.js"
"/IPython/extensions/NewTurtle.py"
"IPython/html/static/notebook/js/myScript.js".

We have created a mime type called "turtle-text" and a function that receives it called "append_turtle" in outputarea.js.

The Turtle commands are sent as a string of characters that are exported when the "run cell" button is pressed. You can get the details in the NewTurtle.py module.

The commands are parsed and drawn using the paper.js library from within the myScript.js file

So far we have implemented the following turtle commands.
backward()
forward()
right()
left()
circle()
penup()
pendown()
speed()
pencolor().

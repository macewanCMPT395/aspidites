Aspidites

IPython Turtle Functionality.

View project onGitHub

Once you've downloaded a copy of Aspidites IPython the first thing you must do to run IPython is open up a terminal window and make the aspidites folder your current working directory. Once you're inside that folder you run IPython with the following command.

$ python -m IPython notebook

You should see IPython open in your default browser. From the dashboard click "New Notebook" then in the first cell you can import turtle just like in normal python using the command below.

from NewTurtle import Turtle

Now we must make a turtle for us to perfom the turtle command with, do this using the line below.

t = Turtle()

Now we're ready to give our turtle some commands.

t.forward(100)

If you hit the play button near the top of the page you should now see the turtle moving forward 100 in the blue canvas below the cell.

Welcome to the official Aspidites IPython Turtle implementation homepage!

For the last 3 months we have been working on integrating a turtle, similar to the popular python educational tool, into the IPython development environment.

The code is available from the links above for you to check out. From within the aspidites directory run

python -m IPython notebook

To use the turtle click on the turtle icon in the top left corner of your notebook or enter the following command.

from NewTurtle import Turtle
t = Turtle()
t.forward(100)

So far you can use the following turtle commands.


backward()
forward()
right()
left()
circle()
penup()
pendown()
speed()
pencolor()

If you want to add to, or modify this project here are a few things that will be useful to keep in mind.

  1. The changes are primarily located in "IPython/html/static/notebook/js/outputarea.js", "/IPython/extensions/NewTurtle.py", "IPython/html/static/notebook/js/myScript.js".

  2. We have created a mime type called "turtle-text" and a function that receives it called "append_turtle" in outputarea.js.

  3. The Turtle commands are sent as a string of characters that are exported when the "run cell" button is pressed. You can get the details in the NewTurtle.py module.

  4. The commands are parsed and drawn using the paper.js library from within the myScript.js file.
